How to Use
  • To add a new node, left-click (Primary Click)
  • To edit an existing node, right-click (Secondary Click)
  • To move a node, drag it on top of another node
  • To delete nodes, click trashcan, then click on a node
  • To add a triangle child, press and hold shift while clicking on a node
  • To draw a movement...fork on Github to help develop this feature
